/// 简介

本实战项目综合运用设计模式板块全部知识，构建一个**模式驱动的迷你游戏框架（MiniEngine）**。项目实现一个终端 RPG 战斗演示——无需图形渲染，重点在架构设计。要求至少运用 8 种设计模式，每种模式必须有明确的设计动机而非"为用而用"。

/// 项目需求

\list{
- 实现终端文字版 RPG 回合制战斗系统
- 角色系统：Builder 构建、组件化属性
- 技能系统：注册式工厂创建、装饰器增强
- 战斗系统：命令模式管理操作、状态机管理阶段
- AI 系统：策略模式实现不同 AI 行为
- 事件系统：观察者模式连接各子系统
- 服务系统：Service Locator 提供全局服务
- 支持完整一场战斗流程和战斗回放
}

// 模式使用清单

\table{
模式 | 分类 | 在项目中的职责
---
注册式工厂 | 创建型 | 从字符串名称创建技能和敌人
Builder | 创建型 | 分步构建复杂角色（名字/职业/属性/技能列表）
装饰器 | 结构型 | 技能增强：增伤、AOE、吸血、附加状态
组合 | 结构型 | 装备/buff 的效果叠加
命令 | 行为型 | 每个战斗操作封装为对象，支持回放
状态机 | 行为型 | 战斗阶段流转（玩家回合→敌人回合→结算→检查胜负）
策略 | 行为型 | AI 行为可切换（攻击型/防守型/随机型）
观察者 | 行为型 | 事件总线通知 UI/日志/计分
Service Locator | 游戏模式 | 提供 Logger、Random、Config 全局服务
ScopeGuard | 现代模式 | buff 过期自动移除
}

/// 技术要求

\list{
- C++17 或更高标准
- 使用 `std::unique_ptr` 管理所有动态分配对象，禁止裸 `new`/`delete`
- 使用 `std::function` 或模板实现事件回调
- 角色属性使用强类型别名（`HP`、`MP`、`Attack`、`Defense` 不能混用）
- 至少 8 种模式正确使用且有意义
- 代码文件组织清晰：每个模式/子系统一个头文件
- 终端输出完整的战斗过程
}

/// 实现提示

// 阶段一：基础架构

\code[cpp]{
// 强类型别名
template<typename Tag>
struct StrongInt {
    int value;
    explicit StrongInt(int v) : value(v) {}
    StrongInt& operator+=(StrongInt o) { value += o.value; return *this; }
    StrongInt& operator-=(StrongInt o) { value -= o.value; return *this; }
    bool operator<=(StrongInt o) const { return value <= o.value; }
    // ... 其他运算符
};

using HP = StrongInt<struct HPTag>;
using MP = StrongInt<struct MPTag>;
using Attack = StrongInt<struct AtkTag>;
using Defense = StrongInt<struct DefTag>;
// HP(100) - Defense(10) → 编译错误！类型不匹配
}

// 阶段二：角色系统

\code[cpp]{
// Builder + Component 构建角色
struct Character {
    std::string name;
    std::string role;
    HP hp; MP mp; Attack atk; Defense def;
    std::vector<std::string> skills;
};

class CharacterBuilder {
    // ... Builder 链式方法
    Character build();
};

auto mage = CharacterBuilder()
    .name("Gandalf").role("mage")
    .hp(HP{60}).mp(MP{200}).atk(Attack{25}).def(Defense{3})
    .skill("fireball").skill("iceblast")
    .build();
}

// 阶段三：技能与战斗

\code[cpp]{
// 工厂 + 装饰器
auto skill = SkillFactory::create("fireball");         // 基础技能
skill = std::make_unique<FireEnhance>(std::move(skill));  // +增伤
skill = std::make_unique<Lifesteal>(std::move(skill));    // +吸血

// 命令模式
class AttackCommand : public ICommand {
    Character& attacker_;
    Character& target_;
    std::unique_ptr<ISkill> skill_;
public:
    void execute() override;   // 计算伤害、应用效果
    void undo() override;      // 恢复状态（用于回放/调试）
    std::string desc() const override;
};
}
